# sing-box-plus

The universal proxy platform.

## Plus Features

- Enhanced TLS Fragmentation (Customizable version from Xray-Core)
- WARP Endpoint Scanner
- WARP Handshake Noise Generator

See the `examples` directory for usage examples.
